# 天气插件（Free API）

本插件基于免费天气接口：`https://aider.meizu.com/app/weather/listWeather`。

功能：
- 启动时自动播报今日天气
- 聊天中询问“天气”时返回实时信息
- 支持插件控制面板与教程入口

使用教程：
1. 打开插件管理，选择天气插件，点击“打开面板”。
2. 在控制面板通过下拉列表选择“城市ID(必填)”，也可手动输入。点击“城市ID列表”可打开对照表。
3. 可选：填写“城市名称(可选)”仅用于显示；“使用 AI 城市”会把 AI 配置城市填入该备注字段。
4. 点击“测试查询”确认能返回天气信息。
5. 若需要启动自动播报，保持“启动自动播报”勾选。

说明：
- 接口无需 API Key。
- 天气查询采用 `cityIds` 参数，示例：`101240101`。
- 若查询失败，请核对城市ID是否正确或稍后重试。
